# 🎟️ Ticket Booking System (Java Console Application)

A **menu-driven Java console application** that simulates a basic ticket booking system.  
Users can create accounts, book tickets, manage journeys, and perform ticket-related operations.

---

## 📌 Features

- 👤 Create User Account
- 📋 View All Users
- 🎫 Book Ticket
- 🧾 View Journey Details by Ticket ID
- 🔍 Search Tickets by User ID
- 🔄 Change Boarding Point
- ❌ Cancel Ticket
- 🗑️ Delete User Account
- 🚪 Exit Application

---

## 🛠️ Technologies Used

- **Java (Core Java)**
- **Collections Framework**
- **OOP Concepts**
  - Encapsulation
  - Abstraction
  - Composition
- **Scanner Class** for input
